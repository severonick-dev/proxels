# @proxels/web

Frontend на Vite + React 18 + TypeScript + TailwindCSS + shadcn/ui. Будет реализован на
Этапе 6 (см. [../../prompts/CLAUDE.md](../../prompts/CLAUDE.md) §12).

Ответственность:

- Лендинг (§3) — мощный дизайн + анимации Framer Motion + SEO (§4c).
- Личный кабинет (§8) — подписка, QR, кнопка ротации subToken, удаление аккаунта.
- Гайды (§9) — пошаговые инструкции для Nekobox/Hiddify/V2RayTun.
- Юр. страницы (§4) и cookie-баннер.
- Темы (dark/light), языки (ru/en), мобильное бургер-меню.
- SSR / pre-render публичных страниц (см. §4c) для SEO.

Везде: единственная соц-сеть — Telegram `https://t.me/proxels`.
